# hoplon

A Leiningen template for creating new Hoplon projects.

## Prerequisites

* Java 8 (not 7)
* Leiningen

## Usage

```bash
$ lein new hoplon my-project-name
```
## Deployment
Just do:
```bash
$ boot prod
```
and copy static files found in your project `/target` directory.
